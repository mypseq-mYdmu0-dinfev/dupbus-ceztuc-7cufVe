## Prerequisite —— use as FALLBACK only

- Public result (e.g. G Scholar) → full text shown → proceed (no library needed).
- Public result → login wall WITH a text input field → type "University of Liverpool" (primary) or "University of Technology Sydney"/"UTS" (secondary) → use the SSO Gate credentials below.
- Public result → login wall with only a drop-down list (no text field) → STOP (institute selection wastes time) → search in the library instead.
- Library result → login wall → use the SSO Gate credentials below.
- In the library (best practice): search the EXACT literature already found in public (e.g. [authors] [year] [title]); do NOT search by topic (use public for that, more efficient).

## Entry Point: UoL (Primary)

- Start at: https://www.liverpool.ac.uk/library/
- Main search bar is the Primo discovery search; redirects to `liverpool.primo.exlibrisgroup.com/discovery/search?...`.

## Search & Results

- Results load on Primo (ExLibris); first result is usually most relevant by default sort.
- Click the article title link to open the full display panel (slides in as an overlay, not a new page).

## Accessing Full Text

- Ignore "For more accurate results please sign in" or similar non-blockers.
- On the panel, locate the **"Get Full Text"** section and click the first link (e.g. "BMJ Journals"); it either opens the full text in a new tab or downloads a PDF.
- If a link fails or leads only to a citation (not full text), return to "Get Full Text" and try the next link; if all fail, flag it and find an alternative.
- NEVER click "Download PDF" if it merely opens the PDF in a new tab (unextractable); NEVER construct a URL to download a PDF (opens in new tab) —— click on-screen buttons/links instead.
- In VitalSource etc., open the ToC (e.g. the `≡` icon) for easier navigation.

## Entry Point: UTS (Secondary)

- Start at: https://www.lib.uts.edu.au —— all operations similar to UoL, except the URL does not show Primo/ExLibris.

## Entry Point: LibKey

- If a result does not show "Full Text Format Options", do not proceed.
- Prioritise "ARTICLE LINK" over "DOWNLOAD PDF" (the latter may open in a new tab, unextractable).
- If struggling, stop and use a direct library search via the entry points above.
