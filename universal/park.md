# Parking Identifier

- If prompted `#park`, follow below procedures
- Goal: Ensure parking near destination (des)
- Parking: Hourly (NOT daily/monthly) car park, street parking, etc.
- Priority: Free > PARK'nPAY (if NSW) > Other
- Vehicle: Camry (if NSW/VIC) or sedan/SUV; never bike/truck

## Procedures

1. Identify des name & address
2. web_search if parking available by official website, social media, etc.
2.1. If no → Step 3
2.2. If yes, verify address in source matches w/ that in Step 1 (not diff entity w/ same name)
2.2.1. If yes → Step 9
2.2.2. If no → Step 3
3. web_search if parking available by unofficial source
3.1. If no → Step 4
3.2. If yes → note address → Step 4
4. Open `maps.google.com` by ACIC (stop if failed)
5. Search des by name (then select from dropdown options, if applicable)
5.1. If failed → try again by address; If still failed → stop
5.2. If succeeded → verify address matches w/ that in Step 1 (not diff entity w/ same name)
6. Open satellite view
6.1. Click `◀︎` icon at vertical centre of right edge of left side panel to collapse it
6.2. Click bottom left box named "Layers"
6.3. Visually locate des again (red pin icon; similar to G Maps logo but in red)
7. Adjust view
7.1. Drag des to horizontal AND vertical centre of view
7.2. Click bottom right `+` icon until max zoomed; usually scale (below `+`; next to "Send product feedback") = 5m
8. Scan for parking
8.1. Prioritise street parking unless you immediately see highly possible free car park nearby
8.1.1. Scan for trail(s) of cars by nearby road sides (NOT road centre, those are moving cars)
8.1.1.1. If none, drag around in all 4 directions whilst keeping the des in view (i.e. nearby)
8.1.1.2. If found, check if the trail(s) form a straight line (not moving cars)
8.1.1.3. If straight, confirm if side-by-side (L parking) or head-to-tail (parallel parking) or other types (e.g. nose-in diagonal)
8.1.1.4. If confirmed → Count how many parked cars (i.e. no. of parking lots) → Step 9
8.2. Fallback: Find car park
8.2.1. Return to map view
8.2.1.1. Click bottom left box named "Layers"
8.2.1.2. Click `▶︎` icon at vertical centre of left edge of whole UI to expand left side panel
8.2.2. Try Westfield (free car park)
8.2.2.1. Search `Westfield` → Locate 1st card → Click circular button w/ `↱` (above "Directions")
8.2.2.2. Check navigation panel opened → Click on input field ("Choose starting point or ...")
8.2.2.3. Type des by name (if Step 5.2 succeeded) or address (if Step 5.2 failed)
8.2.2.4. Locate pedestrian icon (between the bus & bike icons) above the input field of 8.2.2.2
8.2.2.5. Locate the text right below it ("[n] min")
8.2.2.5.1. If n ≤ 10 → Step 9
8.2.2.5.1. If n ≥ 11 → Step 8.2.3
8.2.3. Other car parks
8.2.3.1. Click `×` at top right corner of left side panel to exist navigation
8.2.3.2. Repeat Step 5 (des returns to centre of view)
8.2.3.3. Search `car park` → Locate 1st card → Click circular button w/ `↱` (above "Directions")
8.2.3.4. Repeat 8.2.2.2, 8.2.2.3, 8.2.2.4, 8.2.2.5
8.2.3.4.1. If n ≤ 10 → Click `🌎` icon to access website → Check hourly rate → Step 9
8.2.3.4.2. If n ≥ 11 → Repeat 8.2.3.3 & 8.2.3.4 for next cards (max 5)
9. Report in artefact
9.1. Concise Verdict: Is parking available near des? (max 10 words)
9.2. Tell which steps (don't use Step no., I won't remember) were performed & struggles (if any)
9.3. Summarise result(s) (starting from most feasible option, if multiple)
9.3.1. e.g. If Step 8.1.1.4 performed, how many lots?
9.3.2. e.g. If Step 8.2.3.4.1 performed, estimate my parking cost by inferring parking duration
