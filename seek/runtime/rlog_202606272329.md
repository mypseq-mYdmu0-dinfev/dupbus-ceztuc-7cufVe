## [202606281149] — Protocol Clarification: ⏳_ Rename on Submission Does NOT Require Voiding

- ⏳_ rename after submission → correct; no voiding needed
- ⏳_ file DELETED (no clean AR created to replace it) → Void Rule breach
- File moved between folders (applied/pending/skipped) → Move Rule applies (copy + void original)
