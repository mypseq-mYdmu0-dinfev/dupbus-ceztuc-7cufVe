# Response to "help me edit the Date Added of dissertation_response_202606041852.md to 202606041959"

1. Real root cause (why my first two attempts failed): Finder's **"Date Added"** is the filesystem catalog attribute `ATTR_CMN_ADDEDTIME` (`addedToDirectoryDate`), NOT the `com.apple.metadata:kMDItemDateAdded` extended attribute.
- 1.1. My earlier writes only touched the xattr, which is merely Spotlight's mirror —— so the value Finder displays never moved.
- 1.2. Swift's `URLResourceValues.addedToDirectoryDate` is get-only, so the fix required the lower-level `setattrlist()` syscall (called via Python ctypes) with `ATTR_CMN_ADDEDTIME = 0x10000000` and a `timespec` of the target time.
2. Fixed & verified via Foundation readback:
- 2.1. **Date Added = 04/06/2026 19:59:00 AEST** ✅ (was 04/06/2026 16:21:09 GMT+10)
- 2.2. Date Created also reads 04/06/2026 19:59:00 AEST; Date Modified 05/06/2026 03:01:55 AEST (untouched by this fix).
