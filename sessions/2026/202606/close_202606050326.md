- 2.2. Set it via `setattrlist()` syscall (Python `ctypes`, `ATTR_CMN_ADDEDTIME = 0x10000000`, `struct timespec`); Swift's `addedToDirectoryDate` is get-only so cannot write it.
- 2.3. Script also refreshes the Spotlight xattr (CFDate plist) so both agree.
- 2.4. DAMF expands to "Date Added Manual Fixer" (per user; docstring updated accordingly).
