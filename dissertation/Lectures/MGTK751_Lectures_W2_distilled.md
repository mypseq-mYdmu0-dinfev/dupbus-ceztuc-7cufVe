## W2.1 — Diagnosing Organisational Problems

### MECE Framework
- **Mutually Exclusive, Collectively Exhaustive** — break problem into non-overlapping categories that together cover all possibilities
- Ideal: 3 categories at L2; max 5; each L2 idea must summarise L3 supporting items
- ⚠️ **Unconventional use:** MECE applies beyond problem diagnosis — use it on lit review structure, methods, data collection, and even presentation slides
- Tells you: where to focus, where to dig deeper; makes analysis persuasive to client (people process 3 things better than 9)
- For TPS topic: could structure problem as (A) team cohesion loss / (B) KM governance risk / (C) TP attrition — MECE at L2

### Minto Pyramid
- **Situation → Complication → Solution** (top-down hierarchy)
- Most important idea at top; supporting ideas below in structured layers
- ⭐ **Directly applicable to A2 presentation** — recommended structure per lectures
- Bottom = conclusion; everything above supports it with data

### Inductive vs. Deductive
- **Inductive** (qual): specific observations → broader generalisations/theory; not certain but probable; qualitative territory
- **Deductive** (quant): general theory → specific hypothesis → data → prove/disprove; more certain
- ⚠️ In practice, research is never purely one — but you lean heavily toward one (qual = inductive for us)

### Research Type Matrix
- Problem certainty determines research type:
  - **Exploratory** = unaware of problem ("sales declining, don't know why")
  - **Descriptive** = aware of problem ("who buys our product?")
  - **Explanatory/Prescriptive** = problem clearly defined → use data to test and prescribe
- ⭐ For TPS topic: problem clearly defined → explanatory/evaluative framing is appropriate and expected

### 5-Step Problem Diagnosis
1. Identify real problem (use MECE)
2. What data is needed?
3. Where is that data?
4. How to collect it?
5. What conclusions could be reached?
- Must be threaded by single epistemological approach throughout — this is the "golden thread" applied to methodology
