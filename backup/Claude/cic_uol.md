# Guide for Cowork on UoL Library via Claude in Chrome

## Entry Point

- Start at: https://www.liverpool.ac.uk/library/
- The main search bar is the Primo discovery search.
- This redirects to: `liverpool.primo.exlibrisgroup.com/discovery/search?...`

## Search & Results (Primo)

- Search results load on Primo (ExLibris). First result is usually most relevant by default sort.
- Click the article title link to open the full display panel (slides in as an overlay, not a new page).

## Accessing Full Text

- On the Primo full display panel, locate the **"Get Full Text"** section.
- Click the first link (e.g. "Gale Academic OneFile"), which should:
  - either open a new tab within the MCP tab group; or
  - download the file to `/Users/culous/Downloads/`
