# Buyer's Guide

- Activate if prompted `#buy` OR on any spending activities
- Exc. investment/trading
- Inc. everything else: physical objects, digital asset, software subscription, etc.
- Location (for nearby, if not detected): Merrylands, NSW 2160
- 1st/2nd artefact: bite-size bullets, all #numbered for easy reply

## Procedures

1. Product Identification
1.1. If exact product in context (e.g. iPad Pro [year], Netflix Premium): identify full name (& spec. if noted or necessary; e.g. colour, tier, upgrade)
1.2. Otherwise: identify category (e.g. tablet, streaming service)
2. Use web_search to:
2.1. Research product/category, market outlook, etc.
2.2. Compare considerable rivals/options; if extensive context provided (e.g. I need iPad for task X only, suggest alternate product categories ONLY if meaningful (e.g. Mac Mini suffices task X AND better OR cheaper)
2.3. Find deals, if any (e.g. limited-time offer/promo code)
3. 1st artefact: Preliminary Report
3.1. Current unvalidated findings
3.2. Very briefly on your next steps (see below)
4. Use ACIC (stop if failed) to validate key claims (e.g. prices, deals), check if actually cheapest (any cheaper means?; e.g. Taobao, Temu, Netflix Family; if any, validate by direct visit), then:
4.1. If physical → Step 5
4.2. Otherwise → Step 6
5. Accessibility
5.1. Use ACIC to open `https://www.google.com/shopping`
5.2. Locate & click search bar input field → Search product
5.3. Locate & click "Nearby" (right below search bar, between "Search" & "Deals")
5.4. Choose & visit up to 5 options → Estimate driving distance unless sold by Woolworths/Coles/Bunnings/JB HiFi (weekly pass-by)
5.5. Note 5.4 candidates' details for "Pickup Options"
5.6. Locate & click "Search" (next to "Nearby" of 5.3; i.e. deselect "Nearby" filter)
5.7. Hunt for cheaper options online (priority: free > fast > other delivery); scroll/nav/config filters (e.g. left side bar) as needed, whilst ensuring:
5.7.1. Options are legit, viable, and accurately what being sought of (❌ scope creep)
5.7.2. Accessed websites (NOT webpages) = max. 10 (exc. Google Shopping)
5.8. Note 5.7 candidates' details for "Delivery Options"
5.9. Optional: web_search (not ACIC) again for "Pickup Options" & "Delivery Options" candidates' webpages → Compare against ACIC results (5.5 & 5.8) → Expose if prices went higher/lower than before (web_search = dated results for comparison)
6. 2nd artefact: Final Report
6.1. Verdict: max. 100 words; e.g. "Pickup Options" how far & ready time? "Delivery Options" how much cheaper (in %) than "Pickup Options" (i.e. worth the wait)?
6.2. Tell which steps (don't use Step no., I won't remember) were performed & struggles (if any)
6.3. Summarise result(s) (starting from most feasible option, if multiple)
7. 3rd artefact: Interactive Dashboard (in HTML or other) for top 3–5 options
