# Temporary Workspace

## Preamble
- To be deleted after use
- Anything to keep for good OR to retain git history → suggest where to move
- DON'T read/edit existing folders unless instructed

## Work
- When explicitly told to work in this directory (except `temp_seek/`):
  - Create subfolder: `temp/[temp_folder]/[deadline_YYYYMMDD]_[≤10char_name]/`
  - Subfolders inside (feel free to suggest more or other naming):
    - `input/` —— provided materials (e.g. brief) & prompts (e.g. to Gemini for image generation)
    - `resource/` —— retrieved/generated materials (e.g. research, image)
    - `output/` —— official deliverables (e.g. report) & user's layout files (e.g. Pages, Keynote)
    - `build/` —— everything else (e.g. temp scripts, script I/O)
- Temporary Folders:
  - `temp_seek/` —— for #psl & when requested
  - `temp_int/` —— interviews/trials; [≤10char_name] = org; e.g. ❌ `NameGroupPty` | ✅ `Name`
  - `temp_misc/` —— everything else by default; [≤10char_name] = project/task name
  - `temp_archive/` —— never touch; stores finished temp work when folder clutters; delete ~1yr
